# 사주풀이 — 무료 사주 분석 웹앱

> 생년월일시를 입력하면 **전통 명리학 규칙 기반**으로 사주팔자를 계산하고 해석하는 클라이언트 사이드 웹앱입니다.

**배포 URL**: https://saju-app-iota.vercel.app

---

## 프로젝트 개요

사주풀이는 LLM이나 AI 추측이 아닌, 수천 년간 이어져 온 **전통 명리학 이론을 코드로 직접 구현**한 사주 분석 서비스입니다. 회원가입 없이, 완전 무료로, 브라우저에서 모든 계산이 완결됩니다.

### 핵심 특징

- **규칙 기반 분석**: 같은 사주를 넣으면 항상 같은 결과. 모든 로직을 추적 가능
- **가입 없이 바로 분석**: 로그인, 회원가입, 데이터 저장 없음
- **완전 무료, 무제한**: 횟수 제한, 유료 기능, 광고 없음
- **클라이언트 계산**: 서버에 데이터를 전송하지 않음. 브라우저에서 모든 계산 수행

## MVP 기능

- **만세력 계산**: 생년월일시 → 사주팔자 (년주/월주/일주/시주)
- **오행 분석**: 목(木)/화(火)/토(土)/금(金)/수(水) 분포 시각화
- **십성 산출**: 일간 기준 비견/겁재/식신/상관/편재/정재/편관/정관/편인/정인
- **용신 판단**: 억부법 기반 신강/신약 판정 → 용신/희신/기신 산출
- **해석 제공**: JSON 기반 규칙 조합 해석 텍스트

### 명리학 설계 결정

| 항목 | 결정 | 비고 |
|------|------|------|
| 자시 처리 | 야자시 (23:00~00:00 = 당일) | 현대 명리학 주류 |
| 지장간 | 본기(本氣)만 사용 | MVP 단순화 |
| 용신 판단 | 억부법(抑扶法) | 가장 기본적이고 직관적 |
| 년주 기준 | 입춘(立春) | 명리학 표준 관행 |
| 생시 모름 | 3주 분석 (시주 제외) | 사용자 편의 |

## 하네스 엔지니어링 (AI Agent 협업)

이 프로젝트는 **AI 에이전트(Claude Code)가 코드를 작성하는 하네스 엔지니어링** 방식으로 구축되었습니다.

| 문서 | 역할 |
|------|------|
| `CLAUDE.md` | 에이전트 진입점. 세션 시작 시 자동 로딩 |
| `TASKS.md` | 4 Phase / 18 Task 작업 분해. 완료 조건 체크리스트 |
| `AGENTS.md` | 자율 범위, 코딩 컨벤션, 도메인 규칙, 테스트 규칙 |
| `ARCHITECTURE.md` | 시스템 구조, 모듈 설계, 타입 정의, 데이터 흐름 |

새 에이전트 세션은 `CLAUDE.md` → `TASKS.md` → `AGENTS.md` → `ARCHITECTURE.md` 순으로 문서를 읽고, `TASKS.md`의 미완료 작업부터 이어서 진행합니다.
