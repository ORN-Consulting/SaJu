## 핵심 설계 결정 요약

| 결정 사항 | 선택 | 근거 |
|----------|------|------|
| 자시 처리 | 야자시 (23:00~00:00 = 당일) | 현대 명리학 주류 |
| 지장간 | 본기만 | MVP 단순화 |
| 생시 모름 | 3주 분석 (시주 제외) | 사용자 편의 |
| 날짜 입력 | 셀렉트박스 (연/월/일) | 오입력 방지, 모바일 친화 |
| 생시 입력 | 시진 선택(기본) + 정확한 시간(옵션) | 유연성 확보 |
| 결과 전환 | 스크롤 애니메이션 | 단일 페이지 UX |
| 사주 카드 순서 | 시주→일주→월주→년주 (전통 우→좌) | 전통 방식 존중 |
| 오행 차트 | 레이더/방사형 | 균형 직관적 표현 |
| 해석 문체 | 제목 격식 + 설명 친근 | 신뢰감 + 접근성 |
| 에러 표시 | 인라인 + 해결 방법 안내 | 사용자 친화적 |
