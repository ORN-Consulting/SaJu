# saju-result.md — 사주 분석 결과 화면 명세

> 사주 분석 결과를 사용자에게 보여주는 화면의 기능 명세입니다.
> 입력 화면(`BirthInputForm`)과 같은 페이지에 위치하며, 분석 버튼 클릭 시 스크롤 애니메이션으로 이 영역에 도달합니다.

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| **위치** | 메인 페이지 하단 (입력 폼 아래) |
| **컴포넌트** | `SajuResult.tsx` (컨테이너) |
| **표시 조건** | 분석 실행 완료 후에만 표시 |
| **애니메이션** | 섹션별 순차 페이드인 |

---

## 2. 결과 섹션 구성 (표시 순서)

```
① 입력 요약 바
② 사주팔자 카드 (4장)
③ 오행 분포 (레이더 차트)
④ 십성 분석 (카드에 포함 + 보충 설명)
⑤ 용신 결과 + 종합 해석
⑥ 다시 분석하기 버튼
```

각 섹션은 위에서 아래로 순차적으로 페이드인됩니다.

---

## 3. 섹션별 상세

### 3.1 입력 요약 바

결과 영역 최상단에 사용자가 입력한 정보를 한 줄로 요약합니다.

| 속성 | 내용 |
|------|------|
| **컴포넌트** | `InputSummaryBar.tsx` |
| **표시 형식** | "1990년 3월 15일 (양력) 오시(午時) · 남성" |
| **생시 모름 시** | "1990년 3월 15일 (양력) 시간 미상 · 남성" |
| **스타일** | 작은 글씨, 배경색 구분, 결과 영역 상단 고정 |

---

### 3.2 사주팔자 카드

#### 배치 순서 (전통 우→좌 방식)

```
┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐
│  시주   │  │  일주   │  │  월주   │  │  년주   │
│ (時柱)  │  │ (日柱)  │  │ (月柱)  │  │ (年柱)  │
└────────┘  └────────┘  └────────┘  └────────┘
```

화면 왼쪽부터: **시주 → 일주 → 월주 → 년주**

> 전통 명리학에서는 오른쪽이 년주, 왼쪽이 시주입니다.
> 화면의 왼쪽→오른쪽이 전통의 오른쪽→왼쪽에 대응합니다.

#### 카드 1장의 구성

```
┌──────────────────┐
│     [십성]        │  ← 해당 천간의 십성 (일간 기준)
│                  │
│   갑 (甲)        │  ← 천간: 한글 + 한자
│   ■ 목(木)       │  ← 천간의 오행 + 색상 표시
│ ─────────────── │
│   인 (寅)        │  ← 지지: 한글 + 한자
│   ■ 목(木)       │  ← 지지의 오행 + 색상 표시
│     [십성]        │  ← 해당 지지의 십성 (일간 기준)
│                  │
│     년주(年柱)    │  ← 기둥 라벨
└──────────────────┘
```

#### 카드 상세 속성

| 요소 | 내용 |
|------|------|
| **컴포넌트** | `PillarCard.tsx` |
| **천간 영역** | 한글(한자), 오행 색상 배지, 십성 라벨 |
| **지지 영역** | 한글(한자), 오행 색상 배지, 십성 라벨 |
| **기둥 라벨** | 하단에 "년주(年柱)" / "월주(月柱)" / "일주(日柱)" / "시주(時柱)" |
| **일간 강조** | 일주의 천간(일간)은 특별 강조 스타일 (테두리, 배경색 등) |

#### 오행 색상 매핑

| 오행 | 색상 | HEX (참고용) |
|------|------|-------------|
| 목(木) | 초록 | `#22C55E` (green-500 계열) |
| 화(火) | 빨강 | `#EF4444` (red-500 계열) |
| 토(土) | 노랑/갈색 | `#EAB308` (yellow-500 계열) |
| 금(金) | 흰색/회색 | `#94A3B8` (slate-400 계열) |
| 수(水) | 검정/파랑 | `#3B82F6` (blue-500 계열) |

> 최종 색상은 디자인 단계에서 조정 가능.
> Tailwind CSS 클래스 기준으로 작성하되, 접근성(명암 대비)을 고려할 것.

#### 십성 표시 규칙

- **일간(日干)은 십성 없음** → "일간(日干)" 또는 "나" 로 표시
- 나머지 7글자(천간 3 + 지지 4)는 일간 기준 십성 표시
- 생시 모름(3주 분석)일 때 → 시주 카드 자체를 표시하지 않음 (3장만 표시)

#### 3주 분석 시 (생시 모름)

```
┌────────┐  ┌────────┐  ┌────────┐
│  일주   │  │  월주   │  │  년주   │
│ (日柱)  │  │ (月柱)  │  │ (年柱)  │
└────────┘  └────────┘  └────────┘

"태어난 시간을 알면 시주(時柱)까지 분석할 수 있어요"
```

### 3.3 오행 분포 (레이더 차트)

#### 차트 구성

| 속성 | 내용 |
|------|------|
| **컴포넌트** | `FiveElementsChart.tsx` |
| **차트 유형** | 레이더(방사형) 차트 |
| **축 5개** | 목(木), 화(火), 토(土), 금(金), 수(水) — 고정 순서 |
| **데이터** | 각 오행의 강도 (0~최대값, 정규화) |
| **색상** | 채워진 영역은 반투명 그라데이션 |
| **라이브러리** | Recharts (`RadarChart`) 또는 CSS/SVG 직접 구현 |

#### 차트 아래 보충 정보

```
┌──────────────────────────────────────┐
│         [레이더 차트]                  │
│                                      │
│  목(木) ■■■■□  4개   가장 강한 오행    │
│  화(火) ■■□□□  2개                    │
│  토(土) ■□□□□  1개                    │
│  금(金) ■□□□□  1개   가장 약한 오행    │
│  수(水) □□□□□  0개                    │
│                                      │
│  "목(木)의 기운이 가장 강하고,          │
│   수(水)가 부족한 사주예요."            │
└──────────────────────────────────────┘
```

- 차트 아래에 오행별 개수를 막대 또는 도트로 표시
- 가장 강한 오행과 약한 오행을 강조 라벨로 표시
- 한 줄 요약 문장 (친근체)

### 3.4 십성 분석

십성은 사주 카드 위에 이미 표시되어 있으므로, 이 섹션에서는 **십성 분포의 의미를 해석**합니다.

#### 구성

| 속성 | 내용 |
|------|------|
| **컴포넌트** | `TenGodsAnalysis.tsx` |
| **내용** | 사주에 나타난 십성의 분포 + 각 십성의 의미 해석 |

#### 표시 형식

```
┌──────────────────────────────────────┐
│  십성 분석                            │
│                                      │
│  당신의 사주에 나타난 십성              │
│                                      │
│  ┌──────┐ ┌──────┐ ┌──────┐         │
│  │ 비견  │ │ 식신  │ │ 편관  │  ...    │
│  │  2개  │ │  1개  │ │  1개  │         │
│  └──────┘ └──────┘ └──────┘         │
│                                      │
│  비견(比肩)이 강한 사주                 │
│  "독립심이 강하고 자기 주관이 뚜렷해요.  │
│   주변 사람들과 경쟁 의식이 있을 수      │
│   있지만, 그만큼 추진력도 강해요."       │
│                                      │
│  식신(食神)의 의미                     │
│  "표현력과 창의력이 돋보여요.           │
│   예술적 감각이나 음식에 관심이          │
│   많을 수 있어요."                     │
└──────────────────────────────────────┘
```

- 사주에 등장하는 십성만 표시 (등장하지 않는 십성은 생략 또는 회색 처리)
- 가장 많이 등장하는 십성을 먼저 배치
- 각 십성별 짧은 해석 (2~3문장, 친근체)
- 해석 텍스트는 `data/interpretations/ten-gods.json`에서 로드

### 3.5 용신 결과 + 종합 해석

#### 구성

| 속성 | 내용 |
|------|------|
| **컴포넌트** | `YongshinResult.tsx` |
| **내용** | 용신(用神) 판단 결과 + 종합 사주 해석 |
| **분량** | 1단락, 5~6문장 (중간 분량) |

#### 표시 형식

```
┌──────────────────────────────────────┐
│  용신(用神) 분석                       │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  당신의 용신은                  │    │
│  │  수(水) 💧                    │    │
│  │                              │    │
│  │  일간 강약: 신강(身強)          │    │
│  │  희신(喜神): 금(金)            │    │
│  │  기신(忌神): 목(木)            │    │
│  └──────────────────────────────┘    │
│                                      │
│  종합 해석                            │
│                                      │
│  "갑목(甲木) 일간으로 목(木)의 기운이    │
│   강한 신강한 사주예요. 목의 기운을      │
│   적절히 조절해줄 수(水)가 용신이에요.   │
│   수의 기운은 지혜와 유연함을 상징하는데, │
│   학문이나 연구 분야에서 능력을 발휘할    │
│   수 있어요. 금(金)도 도움이 되는        │
│   희신이니, 가을이나 서쪽 방향이         │
│   좋은 에너지를 가져다줄 수 있어요."     │
└──────────────────────────────────────┘
```

#### 용신 카드 내용

| 항목 | 내용 |
|------|------|
| **용신(用神)** | 필요한 오행 + 아이콘/색상 표시 |
| **일간 강약** | 신강(身強) 또는 신약(身弱) |
| **희신(喜神)** | 용신을 돕는 오행 |
| **기신(忌神)** | 피해야 할 오행 |

#### 종합 해석 규칙

- 분량: 5~6문장, 1단락
- 문체: 친근체 (~이에요, ~해요)
- 포함 내용:
  - 일간의 특성 요약
  - 오행 균형 상태 설명
  - 용신의 의미와 실생활 연결
  - 희신 활용 방법 (방향, 계절, 색상 등)
- 해석 텍스트는 `data/interpretations/yongshin.json` 기반 + 조합 생성

## 4. 애니메이션

### 4.1 섹션별 순차 페이드인

분석 완료 후 결과 영역의 각 섹션이 위에서 아래로 순서대로 나타납니다:

| 순서 | 섹션 | 딜레이 |
|------|------|--------|
| 1 | 입력 요약 바 | 0ms |
| 2 | 사주팔자 카드 | 200ms |
| 3 | 오행 분포 차트 | 400ms |
| 4 | 십성 분석 | 600ms |
| 5 | 용신 + 종합 해석 | 800ms |
| 6 | 다시 분석하기 버튼 | 1000ms |

### 4.2 애니메이션 속성

```
opacity: 0 → 1
transform: translateY(20px) → translateY(0)
duration: 500ms
easing: ease-out
```

### 4.3 사주 카드 개별 애니메이션

4장의 카드도 왼쪽부터 순차적으로 나타나면 더 역동적입니다:

| 카드 | 딜레이 (섹션 딜레이 200ms 기준) |
|------|-------------------------------|
| 시주 | +0ms |
| 일주 | +100ms |
| 월주 | +200ms |
| 년주 | +300ms |

## 5. 반응형 레이아웃

### 모바일 (< 640px)

- 사주 카드: 2×2 그리드 또는 가로 스크롤
- 레이더 차트: 너비 100%, 최소 높이 250px
- 십성 배지: 가로 스크롤 또는 줄바꿈
- 종합 해석: 전체 너비

```
모바일 카드 배치 (2×2):
┌──────┐ ┌──────┐
│ 시주  │ │ 일주  │
└──────┘ └──────┘
┌──────┐ ┌──────┐
│ 월주  │ │ 년주  │
└──────┘ └──────┘
```

### 태블릿 (640px ~ 1024px)

- 사주 카드: 4장 가로 1줄
- 차트와 해석: 1열 배치

### 데스크톱 (> 1024px)

- 사주 카드: 4장 가로 1줄 (충분한 간격)
- 오행 차트 + 십성 분석: 2열 배치 가능
- 최대 너비 제한 (960px)
- 전체 결과 영역 카드 형태로 감싸기

---

## 6. 접근성

- 오행 색상은 색상 + 텍스트 라벨로 이중 표시 (색각 이상 대응)
- 레이더 차트는 `aria-label`로 수치 정보 전달
- 차트 아래에 텍스트 기반 오행 분포 표시 (차트를 볼 수 없는 경우 대비)
- 섹션 간 이동을 위한 키보드 네비게이션 지원
- 애니메이션은 `prefers-reduced-motion` 미디어 쿼리로 비활성화 가능
